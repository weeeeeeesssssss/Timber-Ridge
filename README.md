# Timber Ridge HOA

This repository hosts the Timber Ridge HOA newsletter and covenants documents.

## Contents

- `Newsletters/2026-Spring-Newsletter.html` — current newsletter HTML page
- `Newsletters/WIP/2026-Spring-Newsletter.pdf` — draft or printable PDF version
- `Covenants/` — association covenants and restrictions documents

## GitHub Pages

The site can be published using GitHub Pages from the `main` branch.

If you want the newsletter to appear as a website, configure GitHub Pages in the repository settings to use the `main` branch and the repository root.
